# 第三章：用户搭建指标体系

标签体系：就是分类，对全集的分类，把内容或用户依据统一的规则分到不同的类别中去，类与类之间彼此有联系，这便构成了标签体系

标签划分：横向和纵向。 一、二、三级为主分类标签，四级为业务标签（标签实体）、五级属性标签

<u>要能描述出体系内的具体标签</u>
